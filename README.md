Planr is a minimal, aesthetic planner web app designed for calm and distraction-free task management. It features a clean, notebook-inspired interface with support for both light and dark mode.

## Features

- Minimal and aesthetic UI  
- Add, complete (strike out), and delete tasks  
- Light & dark mode toggle  
- Notebook-style design  
- Fully web-based (no downloads)
- Provides local storage 


## Tech Stack
- HTML
- CSS
- JavaScript
